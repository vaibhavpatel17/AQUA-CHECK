📋 Civil Engineering: Sensor Calibration Protocol

  1. pH Sensor: Calibrate using pH 4.0, 7.0, and 10.0 buffer solutions
  2. TDS Sensor: Use 1000ppm NaCl standard solution for calibration
  3. Turbidity: Use 0 NTU (distilled water) and 100 NTU standards
  4. Temperature: Verify with calibrated thermometer at 25°C
  5. Allow 2 minutes warm-up time before taking readings
  6. Rinse probe with distilled water between different samples

Civil Note: Sensor placement depth and flow velocity affect readings. Ensure representative sampling location.

🔧 Civil Engineering: Arduino Hardware Setup

Required Components

  • Arduino Uno/Nano
  • Analog pH Sensor (Gravity)
  • Turbidity Sensor (TS-300B)
  • TDS Sensor Module
  • DS18B20 Temperature Probe
  • ESP8266 WiFi Module (optional)
  • Waterproof Enclosure (IP65)

Sensor Placement

  • Rivers: 30cm below surface, mid-stream
  • Lakes: Avoid stagnant areas, near inflow
  • Borewells: Pump discharge point
  • Taps: After 2 minutes of flow
  • Secure mounting to prevent movement

Data Transmission

  • USB Serial (direct connection)
  • Bluetooth HC-05 (wireless)
  • WiFi ESP8266 (remote monitoring)
  • JSON format for compatibility
  • 2-second sampling interval
